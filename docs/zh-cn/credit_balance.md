# 查询账户余额

您可以通过[登录控制台](https://console.cloudbypass.com/#/api/)查看账户余额

### 通过HTTP API

通过HTTP API和密钥获取账号实时积分余额，您可以使用任何程序或脚本调用此接口。

```shell
curl "https://console.example.io/api/v1/balance?apikey=<APIKEY>"
```
